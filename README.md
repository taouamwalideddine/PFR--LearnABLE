# PFR--LearnABLE

LearnAble is a specialized learning platform for children with autism. This project uses React (Vite) for the frontend and Express/PostgreSQL for the backend.

## Project Structure
- `/frontend` - React 19 + Vite + Tailwind CSS 4
- `/backend` - Node.js Express + Prisma ORM + PostgreSQL

## Setup
Please check the `database.sql` inside the backend folder for the initial database schema, or run `npx prisma db push` inside the backend directory.
